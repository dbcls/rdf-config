# 実行方法
## ccle
```
# Turtle
bundle exec rdf-config --config work/med2rdf/config/ccle --convert :turtle work/med2rdf/input/ccle

# JSON-LD
bundle exec rdf-config --config work/med2rdf/config/ccle --convert :jsonld work/med2rdf/input/ccle

# JSON Lines
bundle exec rdf-config --config work/med2rdf/config/ccle --convert :jsonl work/med2rdf/input/ccle
```
